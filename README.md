# 🦞 X-Stock 自主交易智体

> **7×24 小时自我进化的 A 股量化交易智能体**  
> 🎯 目标：最大化收益 | 🛡️ 原则：零风险 | 📚 能力：持续学习

---

## 🌟 特性

- **多策略融合**：动量 + 均值回归 + ML 预测 + 情绪分析，4 策略投票决策
- **严格风控**：单票≤20%、单日亏损≤2%、总回撤≤10%、止损 8% 强制卖出
- **免费数据源**：AKShare + Tushare，无需 API Key
- **Web 实时监控**：Streamlit 看板，随时随地查看持仓和收益
- **自我进化**：每日自动复盘、参数优化、GitHub 新策略扫描

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置参数

编辑 `config.yaml`：
```yaml
trading:
  mode: paper  # paper=模拟盘，real=实盘
  initial_capital: 1000000  # 初始资金

risk:
  max_position_per_stock: 0.20
  stop_loss_per_stock: 0.08
```

### 3. 启动服务

```bash
# 启动 Web 看板
python run.py web

# 启动交易引擎
python run.py trade

# 启动所有服务
python run.py all
```

### 4. 访问看板

浏览器打开：`http://localhost:8501`

## 🛡️ 风险控制

- ✅ 单只股票最大仓位：**20%**
- ✅ 单日最大亏损：**2%**（触发后禁止买入）
- ✅ 总资金回撤警戒线：**10%**（触发后停止交易）
- ✅ 单只股票止损线：**8%**（强制卖出）
- ✅ 总仓位上限：**95%**（保留现金应对风险）

## 🧠 自我进化机制

X-Stock 不是一成不变的程序，而是**每天进化的智能体**：

### 每日循环
```
收盘后 (15:00) → 当日交易复盘 → 盈亏分析 → 策略评估
                    ↓
进化时间 (20:00) → 参数优化 → 策略迭代 → 经验沉淀
                    ↓
每周日 (02:00)  → GitHub 扫描 → 新策略集成 → 论文学习
```

### 学习内容
- 📈 每日交易数据 → 优化策略参数
- 🔍 GitHub 顶级量化项目 → 学习新策略
- 📚 最新量化论文 → 吸收前沿方法
- 🌐 市场变化 → 调整模型权重

## ⚠️ 风险提示

1. **模拟盘≠实盘**：滑点、冲击成本未完全模拟
2. **历史不代表未来**：过往策略表现不保证未来收益
3. **黑天鹅事件**：极端行情下策略可能集体失效
4. **技术风险**：网络中断、API 限流等

**建议**：至少运行 3 个月模拟盘，验证稳定性后再考虑实盘。
